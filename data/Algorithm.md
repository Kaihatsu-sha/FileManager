1)Происходит загрузка конфигурации, в случае неудачи происходит инициализация стандартными параметрами.
2)Происходит вывод содержимого каталога по настройкам конфигурации.
 - Настройка постраничного вывода.
 - Вывод информации на экран
 - Если требуется вывод с глубиной больше 1-цы, происходит рекурсивный вызов метода.
3)После вывода содержимого на экран происходит подсветка текущего выбранного элемента с выводом дополнительной информации.
4)Ожидаем ввода пользователя. В зависимости от ввода пользователя:
Изменяет выбранный элемент
Происходит постраничная смена элементов
Переход по файловой структуре
Перевод приложения в считывания команд с клавиатуры.
5) Повторяем 2-4 до ввода команды выхода из приложения.
6)При выходе из приложения происходит сохранение настроек и текущей директории.
